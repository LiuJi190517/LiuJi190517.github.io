Github和Git的区别

  • Git
    版本管理系统（自我理解 源代码管理系统它可以实现本地和远程服务器的同步 可以使用Branch便捷团队做项目 可以进行历史源代码还原等…）
    版本控制是一种记录一个或若干文件内容变化，以便将来查阅特定版本修订情况的系统，对于软件开发领域来说是最重要的一环
  • Github
    基于Git提供的版本管理服务 Github上托管的代码都是基于Git进行版本控制的 Git只是Github上用来管理项目的工具
